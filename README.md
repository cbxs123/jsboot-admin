# jsboot-admin java Spring Boot 项目脚手架

## 介绍

jsboot-admin一个更简洁、更灵活的Spring Boot项目脚手架，致力于帮助开发者快速的搭建项目，让大家更专注于业务实现而非框架问题。

## 特点

集成框架(特色功能)：
- 基于最新的Spring Boot 2
- mysql 
- druid 数据库连接池
- 附带基础数据结构sql
- swagger2 api文档
- redis
- mail
- freemarker
- shiro + redis
- jwt
- mybatis/jdbcTemplate可选
- 定时任务
- 内置用户-角色-权限控制
- 多角色免重登切换
- 模块标签页切换免刷新
